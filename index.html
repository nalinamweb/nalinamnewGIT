How It All Began: The Story of Nalinam

The Spark of Curiosity

With a lifelong fascination for computer science and information systems, I embarked on a mission nearly two decades ago to revolutionize Tamil typing. This led to the creation of Nalinam — the world's first bilingual Tamil word processor with a native-language graphical interface.

The DOS Era: Rewiring the Rules

The initial version of Nalinam was developed during an era dominated by DOS, where hardware and software were not built for Asian languages. I had to bypass system constraints using direct hardware interfacing — rewriting extended ASCII in VGA RAM and creating my own printer and keyboard drivers. It wasn’t just programming; it was engineering.

Enhancing Interaction with Innovation

From basic text entry to complete user engagement, I built a GUI, font utility, interactive tutorials, and even word games. All of this — over 50,000 lines of original C++ code — was shared freely, downloaded by thousands across 50+ countries.

From DOS to Windows

As operating systems evolved, so did Nalinam. I rewrote the engine in 10,000 lines to adapt to Windows, streamlining core features while retiring those made redundant by the OS. Fonts and editors became the focus — simpler, smarter, faster.

Typing Ahead of Microsoft

I then developed a technique to let users type Tamil in any Windows application by manipulating the message queue before the keyboard signal reached the program — something Microsoft later introduced, but never with Nalinam’s intuitive design.

Looking Ahead

With Tamil use growing globally — in schools, governments, and media — I anticipate that large players will catch up. But I’m proud Nalinam was first, and remains a community-driven pioneer.

Partnering with Articulate Multimedia

In the late 1990s, I joined forces with Malaysia’s Articulate Multimedia to build the world’s first Tamil edutainment software. Our project, "Story Time with Arti," won the Best Multimedia Title of the Year 1998. We went on to develop 9 more titles that reached classrooms across Malaysia and Singapore.
