User Manual

How to install - 5 simple steps and you are ready

  1. Right Click on your mouse here
  2. Choose Save Target as to download the self-extractor to any directory by following the instructions
  3. Click the nalinam8.exe that you downloaded to extract the files in the directory.
  4. You should see 5 files on your desktop
  5. Click the fonts.exe to extract the tamil font files.
  6. Click on the Nalinam icon to type your document
  7. If you don’t see tamil, please reboot your computer
  8. You can now copy Nalinam8.exe to pass to your friends.

How to use

  1. Click the Nalinam8 icon
  2. Click OK when you see the peacock.
  3. You should see the Nalinam keyboard now. Study the keyboard or make a printout for reference. Close the "keyboard window".
  4. You are ready to use Nalinam. Just click on the typing window and start typing. If you’re a first time user you need to play around

 

How to type in tamil

  1. Study the Nalinam keyboard layout carefully.
  2. You can type all the basic tamil characters (consonants and vowels) by simply following the arrangements in the layout.
  3. If you need to type any "uyirmei" character, just combine the appropriate "mei" (consonant) and "uyir" (vowel) characters.
  4. If you don't see the following in Tamil, then your tamil fonts are not installed properly. Follow the installation steps above.

How to use in MS-Word and other programs

This version of Nalinam consists of 7 fonts and one tamil/hindu symbols. You can cut and paste tamil words into any other english-based program including MS-Word.

  1. Highlight all the tamil/english words in Nalinam
  2. Click the copy icon in Nalinam
  3. Minimize the Nalinam program
  4. Go to MS-Word or any other program
  5. Click the paste icon. You should see the selected tamil words
  6. You can now use all the features found in that program
